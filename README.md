# SimpleProductAPI

Profesyonel bir ürün yönetim uygulaması. PHP backend REST API ve modern frontend dashboard içerir.

## Özellikler
- REST API (GET, POST, DELETE)
- Ürün listeleme ve ekleme
- Toplam ürün, toplam fiyat ve ortalama fiyat istatistikleri
- Kategori filtreleme ve arama
- Responsive ve modern dark mode tasarım

## Kurulum
1. Veritabanı oluştur: `database/schema.sql`
2. `.env.example` dosyasını `.env` olarak kopyala ve DB bilgilerini gir
3. Web server kökünü proje dizinine ayarla
4. `frontend/index.html` üzerinden projeyi aç

## Yapı
- `api/` → backend
- `frontend/` → frontend CSS & JS
- `database/` → veritabanı şeması
